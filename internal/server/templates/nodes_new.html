New node

Pick how the new node will run. Each option takes you to a form for the type-specific configuration.

Permanent Agent

A Linux, Windows, or macOS box you connect manually by running wolfci-agent on the remote machine. wolfCI pre-registers the slot with the labels and executor count you choose here, then shows you the connection command to copy across.

Google Compute Engine

A pool of ephemeral VMs wolfCI spins up on demand for jobs needing one of the pool's labels. You configure the GCP project, zone, machine type, image, and labels here; the scheduler provisions and terminates instances against that config.

Copy existing node

Duplicate the configuration of a node you already have set up, then edit before saving. Useful for cloning a permanent agent's labels into a sibling box without retyping every field.
